# Expense Tracker Web App

A personal finance tracking application built using Django, SQLite/MySQL, and Chart.js for expense management, budgeting, dashboards, and financial analytics.

The application enables users to monitor transactions, visualize spending trends, and manage recurring expenses through an interactive dashboard.

---

## Features

- Expense tracking and management
- Budget monitoring
- Financial dashboards
- Interactive charts and analytics
- Recurring expense handling
- CRUD operations for transactions
- Export functionality
- Responsive UI
